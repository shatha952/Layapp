# LayApp

LayApp is a web-based application designed to improve retail store layouts and enhance customer flow in R&B stores. The system helps store management visualize store layouts, identify congested areas, collect feedback, and support better layout decisions.

## Project Overview

Retail store layout plays an important role in customer movement, shopping comfort, product visibility, and sales performance. Poor layout design can cause congestion, unclear navigation, low product exposure, and reduced customer satisfaction.

LayApp aims to provide a simple and cost-effective solution for medium-sized retail stores by helping managers and staff analyze customer flow and improve store layout efficiency.

## Main Features

- User login and registration
- Store layout visualization
- Customer flow analysis
- Congestion area identification
- Layout update and control
- Customer feedback collection
- Staff dashboard
- Manager dashboard
- Reports and activity updates
